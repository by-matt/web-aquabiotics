# Aquabiotics Sur: Marine Circular Biorefinery

AquAbiotics is a biotechnology initiative dedicated to the valorization of industrial residues from the marine processing industry. We specialize in the recovery of high-value metabolites through a sustainable circular economy model.

AquAbiotics operates at the heart of the blue economy. Our model creates a closed-loop system where:

- **Molecular Valorization**: Isolate and concentrate endogenous marine metabolites using selective separation technologies.
- **Zero Waste Objective**: Minimizing the organic load (BOD/COD) in industrial discharges to preserve marine ecosystems.
